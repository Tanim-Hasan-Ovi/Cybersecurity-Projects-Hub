## Featured Projects

### 1. 🎣 Phishing URL Detector
A smart web application that leverages Machine Learning to distinguish between legitimate URLs and malicious scams.
* **Live Demo:** https://phishing-url-detector-ovi.streamlit.app
* **Key Capabilities:** AI-powered classification, Fake Brand Detection (Typosquatting), and Trusted Sites Verification.
* **Tech:** Scikit-Learn, Streamlit, Pandas, NumPy.

### 2. 🛡️ Web Vulnerability Scanner & Recon
An automated engine for security researchers to perform rapid audits and intelligence gathering.
* **Live Demo:** https://web-vulnerability-scanner-ovi.streamlit.app
* **Key Capabilities:** Header Audit (HSTS, CSP, X-Frame), Network Port Mapping, WHOIS Intelligence, and Dynamic Risk Scoring.
* **Tech:** Python, Streamlit, BeautifulSoup, Socket.
